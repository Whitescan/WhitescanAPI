# WhitescanAPI
Lightweight Minecraft API for Whitescan's projects

# Java Version
This project uses Java 17

# Minecraft Version
Supported Minecraft (Java 1.18.2) Versions:
- BungeeCord
- Bukkit

# Features
Manage multiple config's without hassle:
  - .yml / .yaml

Manage multiple Databases without hassle:
  - SQLite
  - MySQL
